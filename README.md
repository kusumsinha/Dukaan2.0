🏪 I designed **Dukaan 2.0** — a hyperlocal grocery delivery app that helps people order from trusted neighborhood kirana stores within a 2 km radius.

While quick commerce platforms like Zepto and Blinkit rely on dark stores and warehouses, I explored a different question:
**What if we empowered the 60 million kirana owners already serving our neighborhoods?**

That idea became **Dukaan 2.0** — a mobile app concept designed to bridge local shopkeepers and nearby customers through a seamless, trust-driven grocery delivery experience.

Here’s what I built:

→ **9-screen consumer journey**: store discovery, map view, product browsing, cart, checkout, and live order tracking
→ **8-screen provider journey**: onboarding, dashboard, order management, inventory, and reviews
→ **Complete design system**: colors, typography, reusable components
→ **Clickable end-to-end prototype in Figma**

Some design decisions I’m especially proud of:

• **Map-first home screen** — showing nearby stores first builds trust instantly and gives users location confidence before browsing
• **Auto-applied best coupon** — reduces cognitive load and speeds up checkout
• **3-step vendor onboarding** — simplifies setup for local shop owners and minimizes onboarding drop-offs
• **Hindi-first product labels** — using terms like *आलू* and *टमाटर* makes the experience intuitive for Bharat-first users

In total, this project includes **17 screens**, **20 color styles**, **8 typography styles**, and one cohesive ecosystem built for both customers and kirana vendors.

This was more than a UI exercise — it was an attempt to design for local commerce, accessibility, and trust at the neighborhood level.
